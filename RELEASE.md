# Comptal2 v1.1.0

Première version stable de **Comptal2**, un logiciel de comptabilité moderne et multi-plateforme pour la gestion et l'analyse de vos transactions financières.

---

## Fonctionnalités principales

### Import de données
- Importation de fichiers **CSV** et **Excel**
- Détection automatique des colonnes
- Mapping personnalisable des champs

### Tableau de bord interactif
- Vue d'ensemble avec statistiques financières en temps réel
- Soldes par compte bancaire
- Graphiques interactifs (camemberts, barres)

### Filtres avancés
- Filtrage par compte bancaire
- Filtrage par catégorie
- Filtrage par période (curseur de dates)
- Recherche textuelle dans les transactions

### Gestion des transactions
- Édition et modification des transactions
- Suppression de transactions
- Interface intuitive pour la saisie manuelle

### Analyse financière (Finance global)
- Graphiques détaillés des dépenses par catégorie
- Visualisation des revenus vs dépenses
- Analyses mensuelles et tendances
- **Nouveaux graphiques** pour une analyse plus complète de vos finances

### Pages Entreprise, Association, Gestion et Projet
- Page dédiée **Entreprise** pour suivre les données liées à l'activité professionnelle
- Page **Association** pour la gestion des comptes d'association
- Pages **Gestion** et **Projet** pour organiser et suivre vos projets et contextes de gestion

### Catégorisation automatique
- Classification automatique des transactions
- Règles de catégorisation configurables
- Apprentissage des patterns récurrents

### Multi-comptes
- Gestion simultanée de plusieurs comptes bancaires
- Soldes individuels par compte
- Vue consolidée globale

### Export de données
- Exportation des transactions filtrées au format CSV
- Export personnalisable selon les filtres appliqués

### Interface moderne
- Support du thème sombre
- Interface responsive et intuitive
- Multi-langues (Français / Anglais)

## Installation

### Windows
- [Comptal2-1.1.0-win-x64.exe](Comptal2-1.1.0-win-x64.exe)
- Système requis : Windows 10/11 (64-bit)
- Double-cliquez sur l'installateur et suivez les instructions

### macOS
- **Intel** : [Comptal2-1.1.0-darwin-x64.dmg](Comptal2-1.1.0-darwin-x64.dmg)
- **Apple Silicon (M1/M2/M3)** : [Comptal2-1.1.0-darwin-arm64.dmg](Comptal2-1.1.0-darwin-arm64.dmg)
- Système requis : macOS 10.15 (Catalina) ou supérieur
- Double-cliquez sur le fichier DMG, puis glissez Comptal2 dans le dossier Applications

### Linux
- [Comptal2-1.1.0-linux-x86_64.AppImage](Comptal2-1.1.0-linux-x86_64.AppImage)
- Système requis : Linux (64-bit) avec support AppImage
- **Installation** :
  1. Téléchargez le fichier AppImage
  2. Rendez-le exécutable : `chmod +x Comptal2-1.1.0-linux-x86_64.AppImage`
  3. Double-cliquez sur le fichier ou exécutez-le depuis le terminal : `./Comptal2-1.1.0-linux-x86_64.AppImage`
- **Note** : Les données utilisateur sont stockées dans `~/.config/comptal2/`

---

## Premiers pas

1. **Lancez l'application** après l'installation
2. **Configurez vos comptes et catégories** dans les **Paramètres**
3. **Importez vos fichiers** CSV ou Excel depuis la page **Import**
4. **Consultez le Tableau de bord** pour visualiser vos données
